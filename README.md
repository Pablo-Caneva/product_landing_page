# Ice Cream Product Landing Page

This repository contains the HTML and CSS code for an Ice Cream Product Landing Page. The page is designed to showcase the history, quality, and flavors of the best ice cream in the city, while providing an engaging user experience with a modern and responsive design.

## Features

- **Responsive Design**: The landing page is fully responsive and adapts to various screen sizes, ensuring a great user experience on both desktop and mobile devices.
- **Interactive Navigation**: The page includes a sticky navigation bar with links to different sections, allowing easy access to key content.
- **Embedded Video**: A section featuring an embedded YouTube video to showcase the ice cream-making process.
- **Flavor List**: A comprehensive list of available ice cream flavors, presented in a visually appealing multi-column layout.
- **Email Subscription Form**: A simple form to collect email addresses for updates and promotions.

## Technologies Used

- **HTML5**: For the structure and content of the webpage.
- **CSS3**: For styling the page, including responsive design, layout, and animations.

## Setup

To run this project locally, follow these steps:

1. **Clone the repository**:
    ```bash
    git clone https://github.com/your-username/ice-cream-landing-page.git
    ```
2. **Navigate to the project directory**:
    ```bash
    cd ice-cream-landing-page
    ```
3. **Open `index.html` in your preferred web browser**.
